# Inverse Kinematics of Delta Robot
***
# 使い方 Usage
gitからcloneする

$ git clone https://github.com/EaterK/delta_ik/tree/master/src

１つ目のターミナルで`roscore`を立ち上げる

$ roscore 

２つ目のターミナルで`delta_ik_node`を実行

$ rosrun delta_ik delta_ik_node
